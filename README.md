# Calculator Program

This project is a multi-stage Python program that demonstrates basic programming concepts such as conditional statements, arithmetic operations, and logical checks.  
It is designed as a learning exercise and progresses step by step with increasing functionality.

---

## Stage 1: Basic Calculator

### Description
A simple calculator that takes **two numbers** and an **operator** (`+`, `-`, `*`, `/`) and displays the result.

### Features
- Supports addition, subtraction, multiplication, and division
- Uses `if-elif-else` statements

## Stage 2: Result Check

### Description
Extends the basic calculator by checking whether the result is:
- Positive
- Negative
- Zero

### Features
- Displays the calculation result
- Prints the nature of the result

## Stage 3: Student Grade Calculator

### Description
A program that calculates a student's total marks, percentage, and grade based on marks from **three subjects**.
